Wireframe

This wireframe uses two frameworks which caluculates position of elements. Bootstrap & Scroll Magic. Pay special attention to where Bootstrap classes "row" and "col-md-12" are called. Also note to where the classes "section#" are called as this is important for the wipe on scroll effect. Code placement of these classes are very important as Bootstrap and ScrollMagic both calculate position of elements. Things can look really strange if the calculations of elements are not synced.
